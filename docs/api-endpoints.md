# API Endpoints

## HTML API

### Root

- `Get \` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `GET /api/users/:id`

### Sessions

- `POST /api/session`
- `DELETE /api/session`

### Songs

- `GET /api/songs/:id`
- `GET /api/songs`
- `POST /api/songs`
- `DELETE /api/songs/:id`
- `PATCH /api/songs/:id`

### Comments

- `GET /api/songs/:id/comments`
- `POST /api/comments`
- `DELETE /api/comments/:id`
